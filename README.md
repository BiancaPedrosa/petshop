# Petshop

[![license](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

Aplicação simples de agendamento de serviços (Express + EJS).

Como rodar:

1. Instale dependências:

```
npm install
```

2. Inicie a aplicação:

```
node app.js
```
